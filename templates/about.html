About CardioPredict AI

Project Overview

CardioPredict AI is a web-based heart attack prediction system developed as a final year project for BSc (Hons) Software Engineering at NSBM Green University in partnership with the University of Plymouth.

Technology Stack
  • Programming Language: Python 3.x
  • Web Framework: Flask
  • Machine Learning: Scikit-learn (Random Forest, Logistic Regression)
  • Data Processing: Pandas, NumPy
  • Frontend: HTML5, Bootstrap 5
  • Dataset: UCI Heart Disease Dataset
How It Works

The system accepts clinical parameters such as age, cholesterol, blood pressure, and chest pain type. These inputs are passed through a trained Random Forest machine learning pipeline which outputs a probability score indicating the likelihood of a heart attack.
