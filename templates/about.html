About CardioPredict AI

A web-based heart attack prediction system for BSc (Hons) Software Engineering at NSBM Green University in partnership with the University of Plymouth.

Project Overview

CardioPredict AI is a web-based heart attack prediction system that integrates a trained Random Forest machine learning model within a Flask web application. Users can input clinical parameters and receive real-time risk estimation with SHAP explainability.

The system is designed for educational and research purposes and demonstrates the practical integration of machine learning and web technologies in healthcare applications.

Project Objectives
  • Review literature on cardiovascular disease prediction using ML
  • Analyze and preprocess the UCI Heart Disease datasets
  • Train and evaluate supervised ML classification models
  • Develop a Flask-based web application
  • Create an intuitive UI for clinical data input
  • Ensure reliability through systematic unit testing
