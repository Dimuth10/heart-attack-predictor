About CardioPredict AI

A final year project for BSc (Hons) Software Engineering at NSBM Green University in partnership with the University of Plymouth.

Project Overview

CardioPredict AI is a web-based heart attack prediction system that integrates a trained machine learning model within a Flask web application. Users can input clinical parameters and receive real-time risk estimation.

The system is designed for educational and research purposes and demonstrates the practical integration of machine learning and web technologies in healthcare applications.

Project Objectives
  • Review literature on cardiovascular disease prediction using ML
  • Analyze and preprocess the UCI Heart Disease dataset
  • Train and evaluate supervised ML classification models
  • Develop a Flask-based web application
  • Create an intuitive user interface for clinical data input
  • Ensure system reliability through systematic testing
